# ALINYA Portfolio

Simple static portfolio website built with HTML and CSS.

## Files

- `index.html` — portfolio markup
- `style.css` — page styling

## Usage

Open `index.html` in a browser to view the portfolio.

## Customize

- Replace `profile.jpg`, `project1.jpg`, and `project2.jpg` with your own images.
- Update the `About`, `Skills`, and `Projects` sections to match your experience.
